Installation
====================================

To use exomagpy, install it in the console using pip:

.. code-block:: console
    
    (.venv) $ pip install exomagpy

Requirements
------------------

The packages required to run exomagpy can be found on GitHub `here <https://github.com/quasoph/exomagpy/blob/main/requirements.txt>`_.
